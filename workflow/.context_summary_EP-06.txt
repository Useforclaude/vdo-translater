📊 MAIN TOPIC: WAVE ANALYSIS (การวิเคราะห์คลื่น)

The speaker teaches how to analyze price movements as waves (multiple candlesticks 
combined) rather than individual candlesticks. This provides a "bigger picture" 
view of momentum and trend strength.

═══════════════════════════════════════════════════════════════════════════════

🌊 KEY CONCEPTS:

1. WAVE SHAPES (3 Types):
   - Linear (เส้นตรง): Straight line, steady momentum
   - Exponential (โค้งต้น): Starts slow, accelerates (like accelerating car)
   - Saturation (โค้งปลาย): Starts strong, loses momentum (like braking car)

2. ANGLE/SLOPE ANALYSIS (ความชัน):
   - Measure angles: 45°, 60°, 75°, 90°
   - Steeper = stronger momentum
   - Flattening = losing strength
   - Use angle to predict continuation vs reversal

3. MOMENTUM PATTERNS:
   - Up fast, down slow = Bullish (likely continues up)
   - Up slow, down fast = Bearish (likely continues down)
   - Compare time taken to rise vs fall

4. SIDEWAYS PATTERNS (3 Types):
   - Type 1: Rectangle (กรอบ) - touches both support/resistance
   - Type 2: Triangle (สามเหลี่ยม) - converging lines, doesn't reach edges
   - Type 3: Widening (ปากเปิด) - diverging lines

5. TRIANGLE ANALYSIS:
   - If swing doesn't reach trendline = gap = weakness
   - If angle flattens on each attempt = losing momentum
   - Predict breakout direction by analyzing slopes

6. CHANNEL & SPIKE:
   - Channel: Sideways box pattern
   - Spike: Sudden vertical move
   - Height of spike ≈ height of channel

7. DIVERGENCE (Divergent):
   - Higher High but Lower Low (or equal Low) = losing balance
   - Warning sign of trend reversal
   - Wait for 3rd attempt before trading

🚗 METAPHORS & ANALOGIES:

1. CAR DRIVING (รถ):
   - ขึ้นเขา → climbing hill (uptrend)
   - เหยียบคันเร่ง → step on gas pedal (accelerating)
   - เหยียบเบรก → step on brake (slowing down)
   - ความเร็ว → speed (momentum)
   - เกียร์ → gear (strength level)
   - Example: "Accelerating from 40, 60, 80, 100... building speed"

2. WATER/DAM (น้ำ/ประตูกั้น):
   - ประตูกั้นน้ำ → dam gate (resistance)
   - กดน้ำ → water pressure pushing down
   - เอียงออก → tilting (weakening)
   - ล้ม → collapse (breakthrough)
   - น้ำทะลัก → water bursts through

3. PENDULUM (ลูกตุ้ม):
   - แรงเหวี่ยง → swing force / momentum
   - เหวี่ยงไป → swinging
   - หมดแรง → losing momentum

4. BATTLE/MILITARY (ทหาร):
   - สู้กัน → fighting / battling
   - ต้าน → defending / resisting
   - บุก → attacking / pushing
   - เพลีย → exhausted / tired
   - ส่งทหาร → send reinforcements

5. CLIMBING/TILTING (ปีน/เอียง):
   - ปีนเขา → climbing mountain (steep uptrend)
   - เอียงออก → tilting out (flattening angle)
   - เท → falling over (collapse)
