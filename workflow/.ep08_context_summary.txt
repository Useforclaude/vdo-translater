MAIN TOPIC
====================================
This episode is a comprehensive tutorial on Money Management (MM) in Forex trading, focusing on TWO core concepts:
1. Position Sizing (Lot calculation based on Stop Loss)
2. Risk-Reward Ratio (RR)

The instructor demonstrates through LIVE EXAMPLES using trading software how to:
- Calculate lot size automatically based on % risk per trade
- Set Risk-Reward ratios (1:2, 1:3, etc.)
- Achieve profitability even with 40-60% win rates by using proper MM

====================================
TEACHING STRUCTURE
====================================

**Part 1: Introduction (Segments 0-4)**
- What is MM?
- Why MM matters (can profit even with bad entries if MM is good)

**Part 2: Position Sizing Formula (Segments 5-93)**
- Concept: Risk % of portfolio
- Formula: (Capital × Risk%) ÷ (SL distance × Pip Value) = Lot Size
- Pip Value explanation (different for each pair)
- Using broker calculators (Tickmill, XM, FBS examples)

**Part 3: Risk-Reward Ratio (Segments 94-123)**
- Concept: TP must be X times SL
- Example: 1:2 RR means TP is 2× the SL distance
- Math proof: 40% win rate + 1:2 RR = Profitable
  - Win 40 times × 200 points = 8,000 points
  - Lose 60 times × 100 points = 6,000 points
  - Net profit = 2,000 points

**Part 4: Combined MM Example (Segments 124-161)**
- Simulated 20-day trading:
  - 60% win rate
  - 1:2 RR
  - 5% risk per trade
  - Result: $1,000 → $2,000 (doubled in 1 month)

**Part 5: LIVE SOFTWARE DEMO (Segments 162-315)**
- Installing the MM indicator/tool
- Setting risk % (e.g., 3%)
- Setting RR (e.g., 1:2, 1:3)
- Drawing SL lines → Automatic lot calculation
- Drawing TP lines → Auto-positioned based on RR
- Multiple examples:
  - Buy orders
  - Sell orders
  - Wide SL vs Tight SL (lot size auto-adjusts)
  - Different RR settings (1:2 vs 1:3)
- All examples show: "Risk 3% = $404 loss IF SL hit, REGARDLESS of SL distance"

**Part 6: Conclusion (Segments 316-319)**
- Encouragement to use MM
- "Portfolio can grow even with imperfect system"
- Installation instructions for the tool

5. **Enthusiasm markers**: "เห็นไหมครับ!", "นี่ไง!", "โอเค!"

6. **Sound effects**: จึก, ปุ๊บ, โป๊ะ
